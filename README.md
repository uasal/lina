# Lina
Wavefront sensing and control algorithms and tools being developed in UASAL. 

The recommended installation method is to clone this into your directory of choice and install into the wokring directory with 

```
pip install -e . 
```

If the MagAOX or XWCL software is also installed and available, then the following command will also include the optional MagAOX dependencies that enable functionality with real time tools like SHMIMs and INDI. 

```
pip install -e . --group magaox
```
